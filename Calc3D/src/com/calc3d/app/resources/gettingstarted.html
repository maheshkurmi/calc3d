Getting Started

Adding Elements

Programming and design: Mahesh Kurmi. calc3D is a feature-rich yet easy-to-use data visualization and graph plotting software. To add different elements either click on buttons in toolbar or select from menu>>insert>>...
Add a point in 3D space defined as (x,y,z) to the scene
Add a line joining 2 points in 3D space to the scene
Add a 2D curve(cartesian or polar) defined as y=f(x), or r=f(t)
Add an implicit 2D curve(cartesian or polar) defined as f(x,y)=0, or f(r,t)=0/code>
Add a parametric curve in 3D space defined as x=f(t),y=g(t),z=h(t)) where t is parameter
Add a plane in 3D space defined as ax+by+cz+d=0 to the scene
Add a explicit surface in 3D space defined as z=f(x,y) to the scene
Add an implicit surface in 3D space defined as f(x,y,z)=0 to the scene, not that it may take some time to build, as it uses marching cube algorithm
Add a parametric surface in 3D space defined as x=f(u,v),y=g(u,v),z=g(u,v)) to the scene
Add Premitive e.g. ellipsoid, torous, prism etc.. to the scene

Setting Up canvas and scene

Toggles antialiasing (smooth draw), but note that antialiasing makes rendering slow.
Toggles light Source on/off, to add/remove light sources go to preferences>>render setting tab.
Toggles stereoscopy (3D anaglyph) on/off, click on canvas to change stereoscopic mode.Make sure you are wearing red-cyan glasses.
Resets the view of camera to default.

Remove Elements

Removes selected element from the scene

edit Elements

click on the table to view the selected elements definition/properties. double click to open object Editor dialog. Object Editor dialog is loaded with default object, edit its properties/ definition and click 'OK' to update the element.
