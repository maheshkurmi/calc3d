What is Calc3D?

This program lets you visualize three dimensional mathematical elements , viz Point, vector, line curve and allows to plot and explore 3 Dimensional Graphs of Mathematical Functions.

Source Code is available in Google codes http://code.google.com/p/calc3d/. Feel free to modify and play around. If you add some new features that you think should be a part of the distribution, just send me a note.
